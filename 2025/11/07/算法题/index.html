2024年

题目

已知有向图 G 采用邻接矩阵存储，类型定义如下。

请设计算法：int uniquely(MGraph G) ，判定 G 是否存在唯一的拓扑序列，若是，则返回1，否则返回0.要求如下。

  1. 给出算法的基本设计思想。（4分）
  2. 根据设计思想，采用 C 或 C++ 语言描述算法，关键之处给出注释。（9分）

解答

第一问

  1. 初始化next = -1，count = 0；
  2. 扫描邻接矩阵，当扫到列为全零的点时做如下判断
    • 若next == -1，则记录next为当前列全零点点下标
    • 若next != -1，则return 0；
  3. 扫描完成后，count ++；
  4. 记录now = next，next = -1；
  5. 扫描下标为now的行
    • 若矩阵中当前扫描的值为1，修改当前值为0，并扫描对应的列
      • 若列为全0，则进行如下判断
        • 若next == -1，则记录next为当前列全零点的下标
        • 若next != -1，则return 0；
      • 若列并非全0，则继续寻找下一个值为1的点
  6. 扫描完成后，count++
  7. 对next的值进行判断
    • 若next == -1，则执行8
    • 若next != -1，则返回4
  8. 判断count值是否等于G.numVertices
    • 若等于，则return 1
    • 反之，return 0
