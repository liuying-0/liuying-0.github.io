F3 数字电路基础

通过晶体管搭建门电路

分析门电路

![F3 P1](/images/F3 P1.png)

A B P1 P2 N1 N2 Y
0 0 导通 导通 截止 截止 1
0 1 导通 截止 截止 导通 0
1 0 截止 导通 导通 截止 0
1 1 截止 截止 导通 导通 0

根据上表，两个输入均为0时，电路输出1，否则输出0，逻辑上符合或非运算，故该逻辑门为或非门。

或门

![F3 P2](/images/F3 P2.png)

由逻辑运算可知，在或非门的输出中加入一个非运算，即可得到一个或门。

![F3 P3](/images/F3 P3.png)
